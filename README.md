# choimory-io-architect

- choimory-io 구조를 diagrams(https://diagrams.mingrammer.com)코드로 작성

# 프로젝트 생성

```shell
python3 -m venv venv
source venv/bin/activate
pip install diagrams
touch choimory-io-architect.py
```

# 시작

`source venv/bin/activate`

# 이미지 생성

`python3 choimory-io-architect.py`

# 종료

`deactivate`
